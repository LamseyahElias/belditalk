## On-Page SEO Guidelines

- Include the primary keyword in the title tag, H1, first paragraph, URL, and meta description.
- Add audio examples or short embedded videos where possible.
- Use transliteration consistently, and explain pronunciation clearly.
- Include cultural context so content is more useful than generic phrase lists.
- Add internal links from free articles to the beginner course.
- Create FAQ sections for long-tail searches.
- Use schema markup for courses, videos, FAQs, and breadcrumbs when appropriate.
