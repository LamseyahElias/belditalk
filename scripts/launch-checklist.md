# Belditalk.com Pre-Launch Checklist

Use this checklist before launching Belditalk.com publicly. Assign an owner and due date for each item.

## Brand and Positioning

- [ ] Confirm brand name: Belditalk.com.
- [ ] Confirm founder bio for Elias Lamseyah.
- [ ] Write one-sentence positioning statement.
- [ ] Define primary audience: beginners learning Moroccan Darija.
- [ ] Define secondary audiences: travelers, heritage learners, expats, partners of Moroccans, Arabic learners interested in dialects.
- [ ] Finalize logo, colors, typography, and visual style.
- [ ] Create brand voice guidelines: warm, practical, culturally respectful, direct.
- [ ] Prepare founder headshot and short intro video.

- [ ] Install analytics.
